# TalentTrack UNC: A Unified Management System for Talent Scholars

## Description
TalentTrack UNC is a web-based and mobile-accessible management system designed for the University of Nueva Caceres Talent Scholar Program. The system centralizes and digitalizes key processes such as applicant recruitment, training management, scholarship evaluation, inventory tracking, event participation, and document management.

It solves the problem of manual and fragmented processes by providing a unified platform that improves efficiency, transparency, and data accuracy for administrators, directors, scholars, and applicants.

---

## Features
- Applicant recruitment and screening
- Training scheduling and attendance tracking
- Scholarship monitoring and evaluation
- Inventory and asset management
- Event and engagement tracking
- Document management system
- Role-based access control
- Notifications and reporting system

---

## Tech Stack
- Frontend: HTML, CSS, JavaScript (or JavaFX if desktop-based)
- Backend: Java (Spring Boot) / Node.js (Express)
- Database: MySQL

---

## Installation Guide

### 💻 Run on Desktop (Web)
1. Clone the repository  
   git clone https://github.com/your-username/talenttrack-unc.git

2. Open the project in your IDE  
   (IntelliJ, NetBeans, VS Code)

3. Install dependencies  
   - For Node.js: npm install  
   - For Java: Load Maven/Gradle dependencies  

4. Configure database  
   - Create a MySQL database  
   - Import the SQL file  

5. Run the backend server  

6. Open in browser  
   http://localhost:3000 (or your configured port)

---

### 📱 Run on Android Phone (Mobile Access)

#### Option 1: Same Wi-Fi (Recommended)
1. Connect your **Android phone and laptop to the same Wi-Fi**
2. Run the backend server on your laptop
3. Find your computer’s IP address  
   Example: 192.168.1.5
4. On your Android phone browser, open:  
   http://192.168.1.5:3000
5. The system will load as a **mobile-responsive web app**

---

#### Option 2: Using Emulator (Advanced)
1. Install Android Studio
2. Open Android Emulator
3. Run your backend server
4. Access using:  
   http://10.0.2.2:3000

---

#### Option 3: Convert to APK (Optional Advanced)
- Use tools like:
  - WebView (Android Studio)
  - Capacitor / Cordova
- Wrap your web app into a mobile app
